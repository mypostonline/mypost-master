## Быстрый старт

В .env указать PROPERTY_ID и API_URL

В addresses.js указать команды с ПЛК

В run.js добавить функции
 - getPostState (состояние поста)
 - startOrder (начало программы)
 - endOrder (окончание программы)

```bash
# Установить зависимости
npm install

# Запуск скрипта вручную
node index.js

# Стандартный запуск через pm2
```
